### Custom language translation

#### Purpose:
Add a new translation for the SeleniumLibrary keywords

#### Learning Objectives:

- The selib tool will be introduced
- Trainees will create their own translation file using selib
- Trainees will demo the usage of their custom translation file
- Trainees will learn how to modify the installed selenium translations library
  to include their new translation file

#### Challenge:

Using the selib tool create your own translation file
