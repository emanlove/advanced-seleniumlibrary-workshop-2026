### Setup and Check Environment for SeleniumLibrary Translations

#### Purpose:
Verify one's environment is properly configured for including SeleniumLibrary
keyword translations

#### Learning Objectives:
- Prior to setup trainees will see what errors an improperly/not setup system produces
- Trainees will setup they system to include the robotframework-seleniumlibrary-translation package using pip install


#### Challenge:
Translate a few library keywords and their documentation into your langauge of choice

#### Resources:

Translations for SeleniumLibrary - https://github.com/MarketSquare/robotframework-seleniumlibrary-translation
