### Get (and expand upon) the Library Instance

#### Purpose:
Learn and understand one way to expand the library by getting the library instance

#### Learning Objectives:

- Trainees will see how to create a new library that builds upon the SeleniumLibrary

#### Challenge:
- Create a python library that has a keyword for returning the current browser's capabilities.

#### Resources:

WebDriver instance within the API documentation - https://www.selenium.dev/selenium/docs/api/py/selenium_webdriver_remote/selenium.webdriver.remote.webdriver.html
