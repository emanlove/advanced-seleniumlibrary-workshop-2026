### Review of Selenium WebDriver Architecture and Specify browser version using the options argument

#### Purpose:
The purpose of this exercise is to establish that the browser is configured with options
and the driver with service arguments

#### Learning Objectives:

- Trainees will be able to diagram the parts of the selenium webdriver architecture
- Trainees will be able to recite the mantra "Options is for Browser and Service is for Driver"

#### Resources:

Description and diagram of Selenium 4 architecture - https://www.browserstack.com/guide/architecture-of-selenium-webdriver
